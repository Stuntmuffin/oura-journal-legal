Privacy Policy

This application is designed as a local-first personal analytics tool.

Health data accessed via the Oura API is retrieved only at the explicit request of the user and is stored locally on the user’s device. The application does not transmit, sell, share, or upload user data to any third-party services or external servers beyond the Oura API itself.

No analytics, tracking, advertising, or telemetry services are used.

Authentication credentials and tokens are stored locally on the user’s device using operating system–provided secure storage mechanisms.

Users retain full control over their data and may delete local data at any time.

This application is intended for personal use only.
